Angular Router (ngRoute)

  1. Was ist es?
    • "Offizielles" Routing Modul für AngularJS
    • Gemacht vom AngularJS-Team
    • War früher Teil des AngualarJS-Cores
  2. Was kann es?
    • Einfaches Routing
    • Routing basiert ausschließlich auf URLs
    • Vernküpfung von URLs und Controllern/Templates

UI-Router (ui.router)

  1. Was ist es?
    • Routing Framework für AngularJS
    • Stammt aus der AngularJS-Community
    • Quasi-Standard für komplexes Routing im Rahmen von AngularJS
  2. Was kann es?
    • Organisation von "States"
    • Zusätzlich URL-Routing
    • Named, Nested und Parallel Views
